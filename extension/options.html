About you

Used in every personalized message. Keep it concise, in first person, under 250 words.

Not sure what to write? Let ChatGPT draft it for you.

Copy this prompt, paste it into ChatGPT, replace the [your raw info] block with your resume / LinkedIn About text / anything relevant, then paste the response back into the "About you" field below.

You're helping me set up a tool that writes personalized LinkedIn outreach notes on my behalf. Write me a short profile bio (150 to 250 words) I can paste into the tool. Include:
- My current role and company
- My education (school, degree, expected graduation)
- My core technical stack (3 to 6 main tools/languages)
- 1 to 2 prior roles, briefly
- Any relevant certifications
- The kinds of roles I'm currently looking for

Write it as a tight first-person paragraph or short bullet list. Plain language, no marketing fluff, no buzzwords. Just the bio, no preamble.

OpenAI API key

Your key calls OpenAI directly from this extension. It never leaves your browser.

How to get an OpenAI API key
  1. Open platform.openai.com/api-keys.
  2. Sign in, or create a free OpenAI account.
  3. Click Create new secret key. Give it a name like "ConnectNote AI".
  4. Copy the key (it starts with sk-) and paste it below.
Your API key and profile are saved only to chrome.storage.sync on this device. They are never sent to any server other than OpenAI's API.
